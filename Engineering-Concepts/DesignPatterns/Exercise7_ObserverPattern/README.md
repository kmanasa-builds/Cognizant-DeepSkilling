# Exercise 7 - Observer Pattern

## Objective

Implement the Observer Design Pattern to notify multiple observers whenever a stock price changes.

## Classes

- Stock (Subject Interface)
- Observer (Observer Interface)
- StockMarket (Concrete Subject)
- MobileApp (Concrete Observer)
- WebApp (Concrete Observer)
- Main (Test Class)

## Output

Mobile App: TCS price updated to ₹3500.0
Web App: TCS price updated to ₹3500.0

Mobile App: Infosys price updated to ₹1600.0
Web App: Infosys price updated to ₹1600.0
