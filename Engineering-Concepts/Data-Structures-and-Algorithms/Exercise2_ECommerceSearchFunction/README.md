# Exercise 2: E-Commerce Platform Search Function

## Objective
Implement Linear Search and Binary Search algorithms to search products efficiently.

## Files
- Product.java
- SearchFunction.java

## Algorithms Used

### Linear Search
- Traverses elements one by one.
- Time Complexity: O(n)

### Binary Search
- Works on sorted data.
- Repeatedly divides the search space into half.
- Time Complexity: O(log n)

## Sample Output

Linear Search Result:
Product ID: 103, Name: Keyboard, Category: Accessories

Binary Search Result:
Product ID: 103, Name: Keyboard, Category: Accessories

## Analysis

### Linear Search
- Best Case: O(1)
- Worst Case: O(n)

### Binary Search
- Best Case: O(1)
- Worst Case: O(log n)

Binary Search is more efficient for large sorted datasets.
