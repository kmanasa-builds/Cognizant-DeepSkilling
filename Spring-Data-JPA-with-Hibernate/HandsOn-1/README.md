# Spring Data JPA - Hands-on 1

## Objective
Demonstrate the implementation of Spring Data JPA with Hibernate and MySQL.

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL 8
- Maven
- IntelliJ IDEA

## Database
Schema: ormlearn

Table: country

Sample Data:
- IN - India
- US - United States of America

## Features Implemented
- Spring Boot application
- JPA Entity (Country)
- JpaRepository
- Service Layer
- MySQL Integration
- Retrieve all countries using Spring Data JPA

## Output
Successfully retrieved all countries from the MySQL database.
