# PL/SQL Exercise 3

## Cognizant Digital Nurture 5.0

### Exercise Title
Stored Procedures

### Objective
Implement stored procedures to automate common banking operations using PL/SQL.

## Scenarios Implemented

### Scenario 1: Process Monthly Interest
- Created the `ProcessMonthlyInterest` stored procedure.
- Applied a 1% monthly interest to all Savings accounts.
- Verified the updated account balances.

### Scenario 2: Update Employee Bonus
- Created the `UpdateEmployeeBonus` stored procedure.
- Updated employee salaries based on department and bonus percentage.
- Verified the updated employee salary details.

### Scenario 3: Transfer Funds
- Created the `TransferFunds` stored procedure.
- Transferred funds between two accounts after checking sufficient balance.
- Verified the updated account balances after the transfer.

## Technologies Used

- Oracle Database XE 21c
- Oracle SQL Developer
- PL/SQL
